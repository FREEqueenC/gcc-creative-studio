1. Data We Collect & Authentications

We rely exclusively on secure federated single sign-on (SSO) technologies provided by Google and GitHub via Google Cloud Identity Platform (Firebase Auth compatibility APIs) to register and authenticate users.

  • Federated Sign-In Scopes: We only request basic public profile metadata including your name, email address, and profile picture.
  • GitHub Scopes requested: We utilize standard scopes read:user and user:email.
  • Local Session Tokens: Upon authentication, a secure JWT session token is saved locally on your browser in localStorage to verify authorization headers for active sessions.
