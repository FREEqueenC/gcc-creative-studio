Platform Overview

Aetheris X Creative Studio is a next-generation decentralized multimedia generative suite designed to bridge high-performance generative models with on-chain identity and asset control.

Vertex AI Integrations

State-of-the-art models including Imagen (Image Generation/Upscaling), Veo (High-fidelity Video), and Gemini (Critic feedback and Prompt enhancement).

Multi-Chain Ownership

Minting creative output as NFTs on Ethereum L2 (Base Mainnet) with LEV tokens, or Flow Blockchain using resource-oriented Cadence smart contracts.

Local Development Setup

Follow these quick steps to spin up the local development suite using Docker.

1. Prerequisites

  • Install Docker & Docker Compose
  • Install Google Cloud SDK (gcloud) and run gcloud auth application-default login
  • Set up target Google Cloud project ID

2. Environment Configuration

Configure a local PostgreSQL instance by setting the following database values inside your backend/.env: 

DB_HOST="postgres"
DB_PORT="5432"
DB_USER="studio_user"
DB_PASS="studio_pass"
DB_NAME="creative_studio"
USE_CLOUD_SQL_AUTH_PROXY=false

Multi-Chain Web3 Tokenization

Mint and verify generated media on-chain using two distinct ecosystems:

Base Mainnet (LEV Token)

Ethereum L2 NFT mapping. Uses standard ERC-721 smart contracts. Media files are automatically pinned to IPFS through the Pinata API before triggering smart wallet transactions.

LEV Contract: 0xf61771F3C6c2a59C8C99f7f2Fd04684b7182E340

Flow Blockchain (Cadence)

Uses Flow's resource-oriented Cadence smart contract standards to manage digital collectibles safely on-chain. Uses the Flow Client Library (FCL) for wallet authentication.

Collection Storage Path: /storage/CreativeStudioNFTCollection
